How To Build A Map

1. run create_map.launch
2. run rviz to visualize/check if data is comming in
3. rosbag record -O mylaserdata /scan /tf /rosaria/pose
4. rosparam set use_sim_time true
5. rosrun gmapping slam_gmapping scan:=scan
6. rosbag play --clock <name of the bag that you downloaded / created in step 2>
7. rosrun map_server map_saver -f <map_name>

General explanation:
http://wiki.ros.org/slam_gmapping/Tutorials/MappingFromLoggedData
